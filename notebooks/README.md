# Notebooks

Esta carpeta contiene notebooks Jupyter para análisis exploratorio y experimentación.

## Organización

- **01_exploracion/**: Notebooks de análisis exploratorio inicial de datos
- **02_scraping_paises/**: Notebooks específicos para scraping por país
- **03_analisis_redes/**: Notebooks de análisis de redes y visualizaciones

Los notebooks son para exploración y desarrollo. El código de producción está en `scripts/`.
