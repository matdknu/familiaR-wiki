# Datos Procesados - Relaciones

Esta carpeta contiene información sobre relaciones familiares entre personas.

**Formato:** CSV con columnas:
- source_name: Nombre de la persona origen
- source_url: URL de la persona origen
- target_name: Nombre de la persona destino
- target_url: URL de la persona destino
- type: Tipo de relación (padre/madre, cónyuge, hijo/a, etc.)

Los archivos CSV no se suben al repositorio.
